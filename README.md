# KFC  Sales Dashboard
Dashboard Power BI untuk menganalisis penjualan KFC berdasarkan:
- Cabang (Branch ID)
- Negara (Country)
- Marketing Squad
- Bulan & Tahun

# Tools
- Power BI
- Microsoft Excel

# Dataset
Dataset berisi kolom:
- branch id
- country
- customer
- marketing squad
- month
- sales
- year

## Insight yang ditampilkan
- Perbandingan penjualan antar cabang
- Tren penjualan per tahun & bulan
- Kontribusi tiap marketing squad
- Filter intraktif berdasarkan negra dan tahun

- Power BI:[KFC Dashboard] (dashboard kfc.pbix)
- Excel : [Dashboard CSV] (data kfc.csv)
